Java中BigDecimal的使用

本文最后更新于：2019年10月30日 凌晨

用途

  • 由于浮点数采用二进制系统表示，而二进制无法精确的表示1/10，好比十进制无法精确表示1/3一样。因此，对于很多值浮点数都是采用其能够表示的离目标值近的数来表示，这有可能会在计算中带来不易察觉的误差。为了解决基本数据类型浮点数不能进行精确计算的问题，Java中专门提供了java.math.BigDecimal类，其提供浮点数的精确计算功能。
  • 金融类项目，对于金额的计算很敏感，推荐使用BigDecimal用于数据操作

用例

比较：

//使用compareTo方法
BigDecimal a, b;
a = new BigDecimal("15.64");
b = new BigDecimal("17.45");

if (a.compareTo(b) == -1) {
    System.out.println("a小于b");
}

if (a.compareTo(b) == 0) {
    System.out.println("a等于b");
}

if (a.compareTo(b) == 1) {
    System.out.println("a大于b");
}

if (a.compareTo(b) > -1) {
    System.out.println("a大于等于b");
}

if (a.compareTo(b) < 1) {
    System.out.println("a小于等于b");
}
//使用min方法
BigDecimal bg1, bg2, bg3;
bg1 = new BigDecimal("184.44");
bg2 = new BigDecimal("236");
bg3 = bg1.min(bg2);
System.out.println("最小值is "+ bg3);

加减乘除：

BigDecimal a, b;
a = new BigDecimal("8");
b = new BigDecimal("2");

//加法
System.out.println("a加b="+a.add(b));
//减法
System.out.println("a减b="+a.subtract(b));
//乘法
System.out.println("a乘b="+a.multiply(b));
//除法
System.out.println("a除b="+a.divide(b));
//绝对值
System.out.println("a的绝对值="+a.abs());
//负值
System.out.println("a的负值="+a.negate());

类型转换：

//BigDecimal类型转为String类型（多用于报文传输或数据库字段转换存储）
String str = null;
BigDecimal a = new BigDecimal("0.82");
float f = a.floatValue();
str = Float.toString(f);
System.out.println(str);
